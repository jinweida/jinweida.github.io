java集合框架二

发表于 2017-12-09 | 更新于: 2018-01-10 |

ArrayList

动态数组,ArrayList中的操作不是线程安全的

HashSet

以哈希表的形式存放元素，插入删除速度很快

LinkedList

继承于AbstractSequentialList的双向链表,也可以被当作堆栈、队列或双端队列进行操作,实现java.io.Serializable接口，这意味着LinkedList支持序列化，能通过序列化去传输,是非同步的

Vector

老的动态数组，是线程同步的，效率很低

HashMap

允许null值作为key和value，HashMap的函数则是非同步的，它不是线程安全的,强引用(StrongReference)”,基于“拉链法”实现的散列表

Hashtable

同步的，即它是线程安全的，支持多线程，Hashtable不允许Null作为key和value,是基于“拉链法”实现的散列表

WeakHashMap

弱引用(WeakReference),基于“拉链法”实现的散列表,单线程程序

SortedMap

有序的“键值对”映射接口

TreeMap

是一个有序的key-value集合它是通过红黑树实现的,一般用于单线程中存储有序的映射,TreeMap 实现了Cloneable接口，意味着它能被克隆
