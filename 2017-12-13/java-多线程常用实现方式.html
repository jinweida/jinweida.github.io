java-多线程常用实现方式

发表于 2017-12-13 | 更新于: 2018-01-14 | 分类于 java |


Java中多线程的方式：继承Thread类；实现Runnable接口,实现Callable接口

Runnable比Thread的优势

可以避免由于 Java 的单继承特性而带来的局限；
增强程序的健壮性，代码能够被多个线程共享，代码与数据是独立的；
适合多个相同程序代码的线程区处理同一资源的情况。

继承Thread实现多线程：

Thread也是实现了Runnable接口。

使用Runnable接口实现多线程

run()方法 VS start方法
run方法：是一个普通的方法
start方法：启动一个新线程，并调用run方法

实现Callable接口

java1.5开始提供，java.util.concurrent包
需要实现call方法
可以返回结果（通过 Future），也可以抛出异常
